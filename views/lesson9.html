Creating CSS3 Transitions and Animations in CSS and triggering them with JavaScript

CSS Transition

A change made to an element after a change in state. In the example below the change in state is the hover style attribute. The transition is defined for the base class of the element with its transition-property: background meaning we will be transisitoning the background, its transition-duration: 1s meaning we will be transisitoning between two values for 1 second, and the transition-timing-function: linear meaning we will transition at a linear speed for the defined time. This is an example of a CSS transition trigger, since we will be using the .box:hover CSS class to switch animations.


JS Transition

We can do the same thing using JS instead of the :hover attribute. All we need to do is add the onmouseover and onmouseout Events to our elements and change the background
