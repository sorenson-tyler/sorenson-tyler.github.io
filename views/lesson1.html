Loops

For loop

Iterates for a specific number of cycles. for(var i = 0;i < array.length;i++) {//logic}

For/In loop

Interates through each attribute inside the looped object.

While loop

Interates while a condition is true.

Do/While loop

Preforms an action while the secified boolean expression remains true.

Functions

A set of code designed to preform a specific task. A function is executed when it is invoked. A function is invoked using the following syntax 'functionName'('parametersOfFunction')

Variables

Containers used in JavaScript to hold a value. May be used with arithmatic symbols to perform calculations. A variable is created when it is declared var variableName;. The value of a variable is given using the assignment operator var variableName = value;.

Parameters

The values passed into a function. These values are referenced by variableName and can be used anywhere inside the function's scope.

Arrays

Collection of values stored within a single variable. Syntax var animals = ["Cat", "Dog", "Mouse"];. If we were to access the value cat in the array we could do something like this var cat = animals[0];. We can use that same index reference to change the value inside animals animals[0] = "Kitten";. Arrays are considered to be of type object, meaning we can include any data type we would like inside the same array.

Associative Arrays

Arrays with named indexes. JavaScript does not support arrays with named indexes.
